## Mini Projeto - Hicaro Calliari Bottenberg - SCTEC - ANÁLISE DE DADOS T2

Projeto de Análise de Dados criado para aplicação prática dos conhecimentos adquiridos no curso da SCTEC.

## OBJETIVO

* Importação da base de dados sugerida pelo SCTEC, que encontra-se no kaggle
* Realizar análise exploratória inicial do arquivo para obter um entendimento inicial das informações
* Efetuar a transformação dos dados, utilizando de funções de limpeza dos tipos de dados ( string, int , float e Datetime)
* Remoção de dados Nulos e duplicatas para que influenciem na análise
* Utilizar de métodos para realizar a análise da estatística do dataframe
* Confecção do relatório

## Tecnologias Utilizadas
- Python
- Pandas
- Jupyter Notebook
